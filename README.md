# FlowNotes

**A lightweight VS Code extension for quickly capturing and organizing your thoughts without leaving your editor**

FlowNotes brings the simplicity and visual appeal of sticky notes directly into Visual Studio Code. Stay focused on your code while keeping important reminders, TODO items, and quick references just a click away. Inspired by traditional sticky notes, FlowNotes provides a distraction-free way to manage your thoughts and ideas right within your favorite editor.

## What is FlowNotes?

FlowNotes is a VS Code extension that adds a dedicated notes panel to your Activity Bar, allowing you to create, organize, and manage notes seamlessly. Unlike external note-taking apps, FlowNotes keeps everything integrated into your development environment so you can maintain your workflow without context switching.

## Key Features

### 📝 Easy Note Creation
- Create new notes instantly with the `+ New` button or using the `FlowNotes: New Note` command
- Start typing immediately without complicated dialogs

### 🎨 Visual Organization with Color Coding
- Assign colors to your notes: Yellow, Blue, Green, or Pink
- Use colors to categorize notes by priority, type, or project
- Quickly identify important notes at a glance

### 📌 Pin Important Notes
- Pin critical reminders to keep them at the top of your list
- Ensure urgent tasks are always visible

### 🔍 Quick Search
- Built-in search filter to find notes by content
- Instantly access the information you need

### ✏️ Simple Editing
- Click any note to edit its content
- Changes are saved automatically

### 🗑️ Clean Management
- Delete notes with a single click using the trash icon
- Keep your note list organized and clutter-free

### 💾 Persistent Storage
- All your notes are automatically saved to VS Code's global state
- Your notes persist even after closing and reopening VS Code
- Access your notes anytime without data loss

## How to Use

1. **Open FlowNotes**: Click the FlowNotes icon in the Activity Bar (left sidebar)
2. **Create a Note**: Click the `+ New` button or use the command `FlowNotes: New Note`
3. **Edit**: Click on any note to edit its content
4. **Customize**: Change colors, pin important notes, or search through your notes
5. **Delete**: Click the trash icon to remove notes you no longer need

## Available Commands

- `FlowNotes: New Note` (`flownotes.addNote`) - Create a new note
- `FlowNotes: Refresh Notes` (`flownotes.refreshNotes`) - Manually refresh the notes panel

## Interface

- **Location**: Activity Bar (left sidebar)
- **View Name**: Notes
- **Icon**: FlowNotes icon in the Activity Bar

## Installation

Install FlowNotes directly from the VS Code Extensions Marketplace:
1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "FlowNotes"
4. Click Install

## Requirements

- VS Code 1.109.0 or later
